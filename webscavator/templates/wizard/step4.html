Change the settings

Change the settings of the case you are currently working on by clicking the spanner. This will allow you to go through the setup wizard again and change any steps.

Work on a different existing case

You can also load an old case by clicking on the database icon.

Work on a new case

If you want to work on a new case click on the plus icon. The case you are working on has been saved in a file called [casefile] which you can load later.
